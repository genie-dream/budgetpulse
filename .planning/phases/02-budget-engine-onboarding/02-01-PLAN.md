  <behavior>
    calcVariableBudget(income, fixedExpenses, savingsGoal):
    - (2_000_000, [{amount: 500_000}], 200_000) → 1_300_000
    - (100_000, [], 200_000) → -100_000 (negative allowed)
    - (1_000_000, [], 0) → 1_000_000
    - (0, [], 0) → 0

    getRemainingDaysInPeriod(today, monthStartDay):
    - (new Date('2026-03-25'), 25) → 31 (March 25 – April 24 = 31 days, first day)
    - (new Date('2026-03-24'), 25) → 1 (last day of period, March 24)
    - (new Date('2026-03-01'), 25) → 24 (mid-period: March 1 is 24 days before March 24 end)
    - (new Date('2026-03-10'), 1) → 22 (standard month: March 10, 22 days left incl today)
    - (new Date('2026-02-28'), 31) → 1 (month has no day 31 — period end is Feb 28 or 27, returns at least 1)
    - (new Date('2026-01-31'), 31) → 1 (today IS the last day of period since next period starts Feb 28 → clamp)

    calcDailySurvivalBudget(variableBudget, remainingDays):
    - (1_200_000, 30) → 40_000
    - (1_000_001, 30) → 33_333 (floor, not round)
    - (variableBudget, 0) → 0 (guard against division by zero)

    formatCurrency(amount, currency):
    - (1_200_000, 'KRW') → '₩1,200,000'
    - (1_200, 'USD') → '$1,200'
    - (150_000, 'JPY') → '¥150,000'
    (Uses Intl.NumberFormat with maximumFractionDigits: 0 — no decimals for any currency in this app)

    detectCurrencyFromLocale():
    - navigator.language = 'ja' → 'JPY'
    - navigator.language = 'ja-JP' → 'JPY'
    - navigator.language = 'en-US' → 'USD'
    - navigator.language = 'ko' → 'KRW'
    - navigator.language = 'ko-KR' → 'KRW'
    - navigator.language = 'fr' → 'KRW' (default)
    - typeof navigator === 'undefined' (SSR) → 'KRW'
  </behavior>

    Key implementation notes for getRemainingDaysInPeriod:
    - Use Math.min(monthStartDay, daysInMonth(year, month)) when constructing Date objects to handle months shorter than monthStartDay (e.g. Feb when startDay=31). Helper: daysInMonth(year, month) = new Date(year, month + 1, 0).getDate()
    - "Today" in remaining days calculation: today counts as day 1 (inclusive). So if today IS the last day of the period, return 1 not 0.
    - Period boundary: if today.getDate() >= monthStartDay, period started this calendar month and ends on (monthStartDay - 1) of the next calendar month (clamped to actual days in that month). Otherwise, period started last calendar month and ends on (monthStartDay - 1) of the current calendar month.
    - Always return Math.max(diff + 1, 1) to ensure at least 1 day.

    formatCurrency implementation:
    - Use Intl.NumberFormat with { style: 'currency', currency, maximumFractionDigits: 0 }
    - Locale mapping: KRW → 'ko-KR', USD → 'en-US', JPY → 'ja-JP'
    - Place this function in src/lib/budget.ts (not constants.ts — keep constants.ts for static data only)

    detectCurrencyFromLocale implementation in src/lib/locale.ts:
    - Guard: if (typeof navigator === 'undefined') return 'KRW'
    - const locale = navigator.language || 'ko'
    - if (locale.startsWith('ja')) return 'JPY'
    - if (locale === 'en-US' || locale.startsWith('en-US')) return 'USD'
    - return 'KRW'
